## 📂 Dự Án Chính

### 🌐 [Editorial Desk - Hệ Thống Quản Lý Nhuận Bút Tòa Soạn (Web)](https://github.com/BuirT/DATN-nhuanbutweb)

**MERN Stack | React.js | Node.js | MongoDB | Chart.js**

Hệ thống quản lý toàn diện cho các tòa soạn báo chí, giúp số hóa quy trình từ nhập liệu bài viết đến tính toán nhuận bút và thanh toán.

#### ✨ Tính Năng Nổi Bật:
- 📊 **Dashboard Thống Kê Real-time** - Biểu đồ cột, tròn, theo dõi hoạt động theo khu vực
- ✅ **Quy Trình Duyệt 4 Bước** - Traffic Light Status (Chờ duyệt → Trình lãnh đạo → Đã duyệt → Thanh toán)
- 💰 **Quản Lý Nhuận Bút & Thuế TNCN** - Tính thuế tự động, cảnh báo vượt ngân sách
- 👥 **Quản Lý Tác Giả & Phân Quyền** - Admin, Lãnh đạo, Kế toán, Nhập liệu
- 📄 **Xuất Báo Cáo Excel** - Thống kê toàn cảnh, in bảng kê trình ký
- 🌓 **Dark Mode / Light Mode** - Giao diện responsive, hiện đại

**Tech Stack:** React (Vite) • Node.js • Express • MongoDB Atlas • Mongoose • Axios • Chart.js • xlsx

### 🖥️ [NewsPay - Hệ Thống Quản Lý Nhuận Bút Tòa Soạn (Desktop)](https://github.com/BuirT/DATN-nhuanbutwinform)

**C# | .NET Framework | SQL Server | AI Integration (Ollama)**

Ứng dụng Desktop hiện đại quản lý nhuận bút tòa soạn với tích hợp AI cho kiểm định bài viết và trợ lý kế toán.

#### 🤖 AI Features (Ollama Offline):
- 🔍 **Kiểm Định Bài Viết AI** - Phân tích đạo văn, chất lượng bài viết, điểm số
- 💬 **Trợ Lý AI Chat** - Tư vấn nghiệp vụ kế toán, truy vấn dữ liệu thực tế từ database
- 🔐 **Hoàn Toàn Offline** - Xử lý AI trên máy local, bảo mật dữ liệu nội bộ

#### 📋 Quản Lý Danh Mục:
- Tác giả, Bút danh, Loại báo, Số báo, Tài khoản

#### 💼 Nghiệp Vụ:
- Nhập nhuận bút, Tra cứu, Phiếu chi, Duyệt phiếu chi, Thanh toán
- Báo cáo tổng hợp, chi tiết, công nợ, tổng hợp tháng
- Xuất Excel, Dashboard trực quan

#### 🔐 Bảo Mật:
- Đăng nhập/Đăng ký với mã hóa mật khẩu
- Phân quyền chi tiết (Admin, Biên tập, Kế toán, Phóng viên)

**Tech Stack:** C# • .NET Framework 4.7.2 • Windows Forms • SQL Server • Guna.UI2 • Guna.Charts • ClosedXML • Ollama Qwen2.5 • Newtonsoft.Json
